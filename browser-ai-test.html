Công Cụ Test AI Trên Trình Duyệt

File HTML độc lập để kiểm tra trực tiếp trên trình duyệt: gửi một yêu cầu, thử nhiều model cùng lúc, so sánh chế độ lọc nội dung mặc định / BLOCK_NONE / OFF, và thử gửi dồn để quan sát giới hạn 429 / RPM.

Cấu hình

Với Gemini Proxy, trang sẽ gửi cả safetySettings và safety_settings để dễ kiểm tra proxy có nhận tham số lọc nội dung hay không.
URL https://story-forge-virid.vercel.app/api/proxy là đúng nếu bạn muốn kiểm tra đúng proxy web của app. Nếu mở file HTML bằng file://, trình duyệt vẫn có thể chặn vì CORS tùy vào header từ server.
Hai nút test RPM mới sẽ giúp bạn đo xem giới hạn có vẻ tính theo key chung hay theo từng model.
